# [[智碁資訊]]

## 🏢 公司簡介 (Company Overview)
- **背景**：深耕台灣超過 30 年的自動辨識系統專家，專注於智慧門禁與考勤管理。
- **核心業務**：AI 人臉辨識、指紋辨識、手機行動打卡系統、QR Code 條碼設備及系統整合。
- **主要客戶群**：大型集團與企業（如台塑、遠東集團、華航等），擁有超過 4,000 家企業客戶。
