# 1. 專案背景概述 (Project Context)

## 1.1 專案目標
本專案旨在為**世禾科技股份有限公司 (SHT)** 導入製造執行系統 (MES)，以提升生產管理效率、數據透明度及跨廠區協作能力。

**世禾科技股份有限公司 (SHT)** **總經理：陳學哲** 的目標是，打造一套全世禾集團都能使用的系統（全集團都使用這套系統來管理）。降低員工需要使用多套系統造成的困擾與不必要的重工。


## 1.2 核心組織與合作夥伴
* **業主 (Client)**：世禾科技股份有限公司 (SHT)。
* **系統供應商/合作廠商**：
    * **維宜科技 (WEI YI)**：協助系統開發與導入。
    * **越世實業 (YUE SHI)**：相關硬體或系統支援夥伴。
    * **顧崇德**：有兩家公司的名片，一開始是說維宜科技，但是報價時卻用 越世實業 來進行報價。
* **專案顧問**：顧崇誠 (外部專家顧問，負責指導專案架構與流程優化)。

親屬關係部分：
顧崇誠（顧崇銘的哥哥）（世禾集團總經理_陳學哲_聽顧崇銘的建議，邀請顧崇誠來擔任顧問，完成此案）
顧崇銘（世禾集團，於 大陸-世正 擔任該廠區的總經理職務）
顧崇德（顧崇銘的弟弟）

# 3. 專有名詞對照 (Terminologies)
* **SHT**：世禾科技 (Shi He Tech)。
* **MES**：製造執行系統 (Manufacturing Execution System)。
* **Action Item**：待辦事項，需追蹤負責人與截止日。
* **ERP**：企業資源規劃系統，需與 MES 進行數據交換。
