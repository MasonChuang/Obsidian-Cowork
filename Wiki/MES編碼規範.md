# 📋 SHT-MES 編碼與數據規範 (Encoding Standards)

## 📌 Part Number (P/N) 15碼格式
1. **設備類型**：1 碼 (0-9, A-Z)
2. **設備廠商**：2 碼 (對應廠商代碼)
3. **客戶代碼**：4 碼 (例如：[[台積電]], [[世界先進]])
4. **工件名稱**：4 碼
5. **Tool ID**：2 碼
6. **材質/厚度**：2 碼

## 📌 數據去重與補強邏輯
- **SN 缺失處理**：凡無原始流水號之物件，系統預設代碼為 `ZZZ`。
- **Chamber 標記**：將 Chamber 資訊併入 Work Order (工單) 而非 P/N，以簡化編碼複雜度。
- **跨廠同步**：由 IT 課定期進行全集團編碼對齊。
